Banco de Dados-MySQL

Conheca o sistema de gerenciamento de banco de dados relacional de código aberto amplamente utilizado no mundo da tecnologia, ele oferece uma solução eficaz para armazenar e gerenciar dados de maneira organizada.

Veja como criar um banco de dados, suas tabelas e também como consultar, incluir, excluir e alterar suas informações.

Conteúdo do Curso

  1. Instalação e configuração básica do MySQL;
  2. Criando um banco de dados e suas tabelas;
  3. Incluindo registros nas tabelas;
  4. Consultando os dados incluídos;
  5. Alterando e excluindo dados;
